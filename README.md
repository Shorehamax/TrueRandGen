# Secure Random CLI Generator

`secure_num_gen.py` is a command-line tool for generating cryptographically secure random values using Python’s built-in `secrets` module. It supports both an interactive menu interface and a standard CLI mode suitable for scripting and automation.

The generator relies on the operating system’s cryptographically secure random number generator (CSPRNG). It does **not** use microphones, cameras, location services, or other hardware sensors.

---

# Features

* Cryptographically secure randomness via Python `secrets`
* Interactive menu-driven interface
* Script-friendly CLI mode
* Multiple output formats:

  * Hex
  * Base64
  * Integer
  * ASCII password-style string
* Configurable byte length
* Ability to generate multiple values per execution
* Optional indexed output labeling
* Built-in input validation and safe bounds

---

# Requirements

* Python **3.8 or newer**
* No external dependencies (standard library only)

# Usage

The program supports two operating modes:

1. **Interactive Mode** (menu-driven)
2. **CLI Mode** (argument-based)

---

# Interactive Mode

If the script is run **without arguments**, the interactive interface launches automatically.

```bash
python secure_num_gen.py
```

Example interaction:

```

===========================
Source: Python secrets / OS cryptographic RNG
No microphone, camera, or other permission-gated physical entropy is used.

Output Format
-------------
1. Hex
2. Base64
3. Integer
4. ASCII password-style string
```

The interface will then prompt for:

* Output format
* Byte length
* Number of values to generate
* Whether to prefix results with an index
* Whether to show a configuration summary

# CLI Mode

CLI mode allows the generator to be used in scripts or automated workflows.

Example:

```bash
python secure_num_gen.py --bytes 32 --format hex --count 5
```

Example output:

```
3a5b9d1e...
92fa11c3...
1bc9de72...
...
```

---

# Command Line Options

| Option          | Description                                     | Default |
| --------------- | ----------------------------------------------- | ------- |
| `--interactive` | Launch the interactive menu                     | Off     |
| `--bytes`       | Number of random bytes per value                | 32      |
| `--format`      | Output format (`hex`, `base64`, `int`, `ascii`) | hex     |
| `--count`       | Number of values to generate                    | 1       |
| `--label`       | Prefix each output with an index                | Off     |

Example:

```bash
python secure_num_gen.py --bytes 16 --format base64 --count 3 --label
```

Output:

```
  1: 5Qy1C9Y6cZf8cKcJ3W==
  2: 4Lh9zq3o5zPzM8cQ3A==
  3: 9m9A7xC2K3cP8sJ9Dg==
```

# Output Formats

## Hex

Encodes the random bytes as hexadecimal.

Example:

```
3f9b1e4c5d7a...
```

---

## Base64

Encodes random bytes using Base64 encoding.

Example:

```
Z3JvY3J5cHRvcmFuZG9t
```

---

## Integer

Interprets the generated bytes as a big-endian integer.

Example:

```
89347982374987234987239847239487
```

The integer range depends on the number of bytes used:

```
0 ≤ x < 2^(8 × byte_count)
```

---

## ASCII Password String

Generates a random string using characters from:

* uppercase letters
* lowercase letters
* digits
* punctuation

Example:

```
vA$7!dQ9mZ@1
```

The length of the string equals the selected byte count.

# Security Notes

* Random values are generated using **Python’s `secrets` module**, which uses the operating system’s cryptographically secure random number generator.
* Suitable for:

  * API keys
  * authentication tokens
  * password generation
  * cryptographic salts
  * secure identifiers

This tool is **not a hardware true random number generator (TRNG)**. It relies on the operating system’s CSPRNG.

---

# Limits

To prevent misuse or excessive memory consumption, the program enforces limits:

| Parameter        | Allowed Range |
| ---------------- | ------------- |
| Byte count       | 1 – 1,048,576 |
| Number of values | 1 – 1000      |

```bash
python secure_num_gen.py --bytes 32 --format base64
```

Generate 10 ASCII passwords:

```bash
python secure_num_gen.py --bytes 16 --format ascii --count 10
```

```bash
python secure_num_gen.py --bytes 8 --format int --count 5 --label
```
